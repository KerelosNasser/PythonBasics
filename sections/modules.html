Modules & Packages — Sharing and Organizing Code
The Import Mechanism

Packages
Standard Library Imports
# Method 1: Import everything
import math
print(math.sqrt(16))  # 4.0

# Method 2: Import specific tools (cleaner)
from random import randint, choice
print(randint(1, 10))

# Method 3: Use an alias (short name)
import datetime as dt
print(dt.datetime.now())
Your Own Modules

Import code from your other `.py` files in the same folder.

# Inside main.py:
import helpers

helpers.say_hi()

# Inside helpers.py:
def say_hi():
    print("Hello!")
Common Useful Modules
  • 📁 os: Move files, create folders
  • 🕒 time: Pause code (`time.sleep`)
  • 🔢 statistics: Mean, Median, Mode
  • 🌐 requests: Get data from websites
  • 📊 json: Convert text to data
The PIP Package Manager

Use the terminal to install thousands of free libraries.

pip install requests
pip install pandas matplotlib

Run these in your command prompt, not inside Python!
