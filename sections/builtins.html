The "Enumerate" Trick

Use when you need the index AND the item in a loop.

names = ["Kero", "Mina", "Peter"]

for i, name in enumerate(names):
    print(f"{i}: {name}")

# Output:
# 0: Kero
# 1: Mina
# ...
The "Zip" Trick

Loop through two lists at the same time.

users = ["Alice", "Bob"]
levels = [5, 12]

for user, lvl in zip(users, levels):
    print(f"{user} is level {lvl}")
Input & Type Casting
# Get absolute value (removes negative)
abs(-10)    # 10

# Round to nearest integer
round(3.1415, 2) # 3.14

# Check if things are true
any([False, True]) # True (any one)
all([True, True])  # True (all of them)
