Variables & The 4 Core Types
Defining Variables & Types
# Python is dynamically typed (no need to declare int/string)
name = "Alice"     # string (str)
age = 30           # integer (int)
height = 5.8       # float
is_student = False # boolean (bool)

# Check type
print(type(age))   # <class 'int'>
Input / Output & Type Casting
# input() always captures data as a string
age_str = input("Enter your age: ") 

# Casting: Convert string to integer
age_int = int(age_str)

# Doing it in one line
price = float(input("Enter price: "))

# Output to screen
print("You are", age_int, "years old.")
