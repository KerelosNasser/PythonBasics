Fundamentals — Numbers & Variables
Basic Data Types
age = 20           # int
price = 19.99     # float
name = "Kero"       # str
is_pro = True     # bool

# Check type of any variable:
type(age)  # <class 'int'>
Type Casting (Conversion)

Convert one type to another using constructor functions.

# String to Int
x = int("10")    # 10

# Int to Float
y = float(5)     # 5.0

# Anything to String
z = str(3.14)    # "3.14"

# ⚠️ Be careful: int("abc") will crash!
Arithmetic & Console I/O
Operators & Math
10 + 3   # 13 (Add)
10 - 3   # 7  (Sub)
10 * 3   # 30 (Mult)
10 / 3   # 3.33 (Div)

# Advanced Math
10 // 3  # 3  (Floor Div - drops decimal)
10 % 3   # 1  (Modulo - remainder)
2 ** 3   # 8  (Exponent - 2 to the power of 3)
Input & Output
# Output to console
print("Hello", 123)

# Get user input
# ⚠️ input() ALWAYS returns a string!
name = input("What's your name? ")
age = int(input("Enter age: ")) # Cast to int immediately
