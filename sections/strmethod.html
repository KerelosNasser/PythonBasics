Deep Dive: String Methods Cheat Sheet
String Transformation Map
Cleaning & Padding
s = "  data  "

s.strip()     # "data" (removes both sides)
s.lstrip()    # "data  " (removes left only)
s.rstrip()    # "  data" (removes right only)

# Padding / Alignment
s.center(12, "*")  # "**  data  **"
s.ljust(12, "-")   # "  data  ----"
"42".zfill(5)      # "00042" (zero-fill)
Searching & Validation
msg = "Python is good"

# Search methods
msg.find("is")      # 7 (index of first 'is')
msg.find("Java")    # -1 (NOT found — safe)
msg.index("good")   # 10 (crashes if NOTfound)
msg.count("o")      # 3 (how many 'o's)

# Validation methods (True/False)
msg.startswith("P") # True
msg.endswith("!")   # False
"123".isdigit()     # True
"abc".isalpha()     # True
Replacement & Splitting
text = "the-quick-brown-fox"

# Replace
text.replace("-", " ") # "the quick brown fox"
text.replace("-", "_", 2) # replace only 1st two

# Split (Turns string into list)
words = text.split("-")   # ["the","quick","brown","fox"]
lines = "A\nB\nC".splitlines() # ["A","B","C"]

# Join (Turns list into string)
fruits = ["apple", "pear"]
" & ".join(fruits)      # "apple & pear"
Case Transformations
msg = "hello WORLD"

msg.lower()      # "hello world"
msg.upper()      # "HELLO WORLD"
msg.capitalize() # "Hello world"
msg.title()      # "Hello World"
msg.swapcase()   # "HELLO world"
