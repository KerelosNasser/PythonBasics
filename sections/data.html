The 4 Built-In Collections
List, Tuple, Set, Dictionary Compared
List Manipulation
heroes = ["Batman", "Superman"]

# Add items
heroes.append("Flash")      # Adds to the end
heroes.insert(0, "Wonder")  # Inserts at index 0

# Remove items
heroes.remove("Superman")   # Removes by value
last = heroes.pop()         # Removes AND returns last item

# Other useful methods
heroes.sort()     # Sorts alphabetically
length = len(heroes)  # Get size of list
Dictionary Manipulation
player = {
    "name": "Kero",
    "hp": 100
}

# Accessing (Safe vs Unsafe)
print(player["name"])          # "Kero" (crashes if key missing)
print(player.get("mp", 0))     # 0 (safe! returns default if missing)

# Adding / Updating
player["level"] = 5   # Adds new key
player["hp"] = 90     # Updates existing key

# Deleting
del player["level"]

# Iterating over dictionary
for key, val in player.items():
    print(key, "->", val)
