Control Flow — Decision Trees
If / Elif / Else
# Indentation (spaces) defines the code blocks!
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")  # This block runs
else:
    print("Grade: F")
Logical Operators (and, or, not)

Combine multiple conditions together.

age = 20
has_id = True
is_vip = False

# 'and' requires BOTH to be True
if age >= 18 and has_id:
    print("Enter club.")

# 'or' requires AT LEAST ONE to be True
if is_vip or age >= 21:
    print("Free drinks!")

# 'not' reverses the boolean
if not is_vip:
    print("Pay entry fee.")
Iteration — For & While Loops
For Loops & Range
# range(stop) -> starts at 0
for i in range(3):
    print(i)  # 0, 1, 2

# Iterating over a list directly
fruits = ["apple", "banana"]
for fruit in fruits:
    print(fruit)
Demystifying range(start, stop, step)
# range(start, stop, step)
r = range(2, 10, 2)
list(r)  # [2, 4, 6, 8]

# Counting backwards
range(5, 0, -1) # 5, 4, 3, 2, 1
While Loops & The Do-While Hack
# Standard while loop (checks before running)
x = 0
while x < 3:
    print(x)
    x += 1  # MUST update condition to avoid infinite loop!

# DO-WHILE SIMULATION (Python has no native do-while)
# Runs AT LEAST once, then checks condition.
while True:
    cmd = input("Type 'quit' to exit: ")
    if cmd == "quit":
        break  # break immediately stops the loop
