Match / Switch-Case (Python 3.10+)
The match Statement — Python's Switch-Case
Basic match — Replacing if/elif Chains
# Old way (messy if/elif chain)
if day == "Mon": print("Start of week")
elif day == "Fri": print("Weekend!")
else: print("Midweek")

# New way — match statement (Python 3.10+)
match day:
    case "Mon":
        print("Start of week")
    case "Fri" | "Sat" | "Sun":  # | = OR
        print("Weekend vibes!")
    case _:             # _ = default (catch-all)
        print("Midweek grind")
Capture Patterns & Guards
# Capture pattern: stores matched value in variable
score = 85
match score:
    case n if n >= 90:  # Guard: extra condition after 'if'
        print(f"A grade: {n}")
    case n if n >= 80:
        print(f"B grade: {n}")  # Prints this!
    case _:
        print("Keep studying")

# Matching sequences (lists/tuples)
point = (0, 5)
match point:
    case (0, 0): print("Origin")
    case (0, y): print(f"On Y-axis at y={y}")
    case (x, 0): print(f"On X-axis at x={x}")
    case (x, y): print(f"Point at {x},{y}")
Matching Dictionaries & Classes
# Match against dict shape (great for API responses!)
event = {"type": "click", "button": "left"}

match event:
    case {"type": "click", "button": btn}:
        print(f"{btn} click!")  # "left click!"
    case {"type": "keypress", "key": k}:
        print(f"Key pressed: {k}")
    case _:
        print("Unknown event")

# match vs if/elif: use match when comparing
# the SAME variable against multiple exact values
Real-World CLI Command Router
# Classic use-case: routing commands in a game/CLI
def handle_command(cmd):
    match cmd.split():         # Split "go north" → ["go","north"]
        case ["quit"]:
            print("Goodbye!")
        case ["go", direction]:
            print(f"Moving {direction}")
        case ["pick", "up", item]:
            print(f"Picked up {item}")
        case [unknown, *rest]:   # *rest = remaining words
            print(f"Unknown: {unknown}")

handle_command("go north")    # Moving north
handle_command("pick up sword") # Picked up sword
