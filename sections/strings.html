String Indexing & Manipulation
How Python Views Strings (Indexing & Slicing)
Slicing & f-Strings
word = "PYTHON"

# Slicing syntax: [start:stop:step]
# * stop index is NOT included!
word[0]      # "P" (first letter)
word[-1]     # "N" (last letter)
word[0:2]    # "PY" (gets index 0 and 1)
word[:4]     # "PYTH" (from start to 3)
word[2:]     # "THON" (from 2 to end)
word[::-1]   # "NOHTYP" (reverses string!)

# Modern F-Strings (Formatting)
# Put 'f' before the quotes, use {} for variables
name = "Kero"
age = 20
print(f"My name is {name} and I am {age}.")
Essential String Methods

Methods are built-in actions you can perform on objects. Strings have many!

msg = "  hello world  "

# Cleaning & Changing Case
msg.strip()     # "hello world" (removes spaces)
msg.upper()     # "  HELLO WORLD  "
msg.title()     # "  Hello World  "

# Searching & Replacing
msg.replace("o", "0") # "  hell0 w0rld  "
"world" in msg        # True (membership check)

# Splitting & Joining
csv = "apple,banana,cherry"
fruits = csv.split(",") # Turns string into list
" - ".join(fruits)     # "apple - banana - cherry"
